# tks-tools-updates

TKS CREATE が販売している Premiere Pro 用 CEP プラグイン群の **バージョン情報配信**用 public リポジトリ。

各ツール本体（ZXP・ソースコード）は **このリポジトリには含まれません**。各ツール本体は別の private リポジトリで管理されています。

## 仕組み

各ツールのパネルは起動時に対応する `version.json` を取得し、自身のバージョンと比較して、新版があれば「🔔 更新があります」ボタンを表示します。

- 送るのはバージョン確認の GET のみ
- お客様の素材・カット結果・API キー・ログ等は一切送信されません
- 通信失敗時は静かに無視（ボタンを出さないだけ）

## 構造

```
tks-tools-updates/
├── README.md
└── muoncut/
    └── version.json    無音カットするだけ
```

将来、文字起こしするだけ等の新ツールが加わると、対応するディレクトリ + `version.json` をここに追加します。

## version.json の形式

```json
{
  "version": "2.0.5",
  "note": "任意の告知文（空可）"
}
```

## リリース運用

新バージョンを BOOTH / STORES に公開したタイミングで、対応する `version.json` の `version` を書き換えて commit + push します。これにより既存ユーザーが次にパネルを起動したときに更新通知が出ます。
